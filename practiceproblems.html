Systems of Linear Equations

Solve the following system of equations using Gaussian Elimination:

2x + 3y - z = 5
x - 2y + 3z = 4
3x + y + z = 10

Hint: Reduce to Row-Echelon Form, then Reduced Row-Echelon Form.

Find the general solution for the following homogeneous system:

x + 2y + 3z = 0
2x + 5y + 6z = 0
4x + 8y + 12z = 0

Hint: Analyze for trivial and non-trivial solutions based on rank and free variables.

Determine the rank and consistency of the following matrix:

[1  2  3  7]
[2  4  6  14]
[3  6  9  21]

Hint: Use Gaussian elimination to determine rank and check for consistency.

Matrix Operations

Perform the matrix addition for the following matrices:

A = [1  2]
    [3  4]

B = [5  6]
    [7  8]

Hint: Simply add corresponding elements of both matrices.

Calculate the matrix product \( AB \):

A = [1  2  1]
    [0  1  2]

B = [3  1]
    [2  4]
    [0  5]

Hint: Row X, Column Y of C is the dot product of Row X of A and Column Y of B.

If \( k = 3 \), multiply matrix A by scalar k:

A = [1  2  3]
    [4  5  6]

Hint: Multiply each element of matrix A by the scalar.

Matrix Inverses

Find the inverse of the following matrix (if it exists):

A = [1  2]
    [3  4]

Hint: Use the formula for the inverse of a 2x2 matrix: \( A^{-1} = \frac{1}{\det(A)} \begin{bmatrix} d & -b \\ -c & a \end{bmatrix} \)

Linear Transformations

Apply a rotation of \( \theta = 90^\circ \) counterclockwise to the vector \( \begin{bmatrix} 1 \\ 0 \end{bmatrix} \).

Rotation Matrix: 
[cos(θ)  -sin(θ)]
[sin(θ)   cos(θ)]

Hint: Use the rotation matrix with \( \theta = 90^\circ \) and calculate the resulting vector.

Matrix Determinants

Find the determinant of the following matrix using cofactor expansion:

A = [1  2  3]
    [4  5  6]
    [7  8  9]

Hint: Expand along the first row and use the cofactor method to compute the determinant.
