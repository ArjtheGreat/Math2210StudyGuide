Systems of Linear Equations

Problem 1: Solve the following system using Gaussian Elimination:

2x + 3y - z = 5
x - 2y + 3z = 4
3x + y + z = 10

Reduce to Row-Echelon Form, and then to Reduced Row-Echelon Form. The solution is:

x = 1, y = 2, z = 3

Problem 2: Prove that a homogeneous system always has the trivial solution.

A homogeneous system is of the form \(Ax = 0\), and the zero vector is always a solution because \(A(0) = 0\). The system can also have non-trivial solutions if the rank of A is less than the number of variables (i.e., there are free variables).

Problem 4: Prove that matrix multiplication is associative.

To prove that \( A(BC) = (AB)C \), show that for any matrices A, B, and C, the corresponding element at row i and column j is the same for both sides of the equation by expanding the dot product definition.

Matrix Inverses

Problem 5: Find the inverse of the following matrix:

A = [1  2]
    [3  4]

Using the 2x2 inverse formula, \( A^{-1} = \begin{bmatrix} -2 & 1 \\ 1.5 & -0.5 \end{bmatrix} \).

Problem 6: Prove that the inverse of a matrix is unique.

Suppose \( A \) has two inverses, \( B \) and \( C \), such that \( AB = I \) and \( AC = I \). Then, \( B = B(AC) = (BA)C = IC = C \). Hence, the inverse is unique.

Problem 8: Prove that linear transformations preserve vector addition.

For a linear transformation \( T \), show that \( T(u + v) = T(u) + T(v) \) by expanding the transformation definition. Since the transformation applies individually to both vectors, the property holds.
